# 🌤️ 天气查询技能

PyClaw天气查询技能提供全球城市的实时天气信息查询功能，支持多种温度单位和查询格式。

## 🎯 功能特点

### 实时天气查询
- 查询指定城市的实时天气信息
- 支持全球主要城市和地区
- 包含温度、湿度、风速、风向等详细信息

### 灵活的查询选项
- **温度单位**：支持摄氏度(°C)和华氏度(°F)
- **查询格式**：多种输出格式，适应不同使用场景
- **响应速度**：快速响应，支持并发查询

## 🛠️ 工具列表

### get_weather - 获取城市天气信息
```json
{
  "name": "get_weather",
  "description": "查询指定城市的当前天气、温度、湿度等信息",
  "parameters": {
    "city": "城市名称（如 '北京'、'上海'、'Tokyo'、'New York'）",
    "unit": "温度单位：'c' 摄氏度（默认），'f' 华氏度"
  },
  "required": ["city"]
}
```

## 📝 快速使用示例

### 查询中文城市天气
```python
from pyclaw import Gateway

gateway = Gateway("api_key")

result = gateway.ask("查询北京的天气")
# 或使用工具调用
result = await gateway.call_tool("get_weather", {"city": "北京"})
print(result)
```

### 查询国际城市天气
```python
result = await gateway.call_tool("get_weather", {"city": "Tokyo"})
print(result)
```

### 指定温度单位
```python
# 查询纽约天气，使用华氏度
result = await gateway.call_tool("get_weather", {
    "city": "New York",
    "unit": "f"
})
print(result)
```

## 🔧 配置说明

### API接口
- 使用 wttr.in 免费天气 API
- 无需额外配置，直接调用即可
- 支持 IPv4 和 IPv6

### 网络要求
- 需要网络连接才能查询天气
- 默认超时时间：15秒
- 重试机制：失败后自动重试一次

## 🐛 故障排除

### 常见错误及解决方案

#### 城市未找到
```
错误: 未找到该城市的天气数据
解决方案: 检查城市名称是否正确，或尝试使用拼音、英文名称
```

#### 网络连接失败
```
错误: 查询天气失败: 连接超时
解决方案: 检查网络连接，或尝试使用其他网络
```

#### 服务不可用
```
错误: 查询天气失败: 服务暂时不可用
解决方案: 稍后重试，或检查 wttr.in 服务状态
```

## 📚 技术细节

### API响应字段
- **城市名称**：查询的城市名称
- **天气描述**：文字描述的天气情况（如多云、晴、小雨等）
- **温度**：当前温度
- **湿度**：空气湿度（百分比）
- **风速**：风速（公里/小时）
- **风向**：风向（方位）
- **气压**：大气压（百帕）
- **能见度**：能见度（公里）

### 单位说明
- **温度**：°C 或 °F
- **风速**：公里/小时
- **气压**：百帕（hPa）
- **湿度**：百分比（%）
