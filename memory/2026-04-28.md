- 系统默认装了 Wine 10.0，用户需要 9.0-1
- 降级操作：`apt install wine/wine64/libwine=9.0~repack-4build3`
- 成功降级到 Wine 9.0

### 中文字体配置
- 安装字体包：fonts-wqy-microhei、fonts-wqy-zenhei、fonts-noto-cjk
- 注册表配置：字体替换（宋体→文泉驿微米黑等）、ClearType 抗锯齿
- 创建 wine-zh 自动设置中文 locale 的启动脚本
- 配置 Bash 别名（wine/winecfg/regedit 自动中文）
- Bottles 里添加 LANG=zh_CN.UTF-8 环境变量
- 最终问题：部分字体还是"口口"（不全），导入完整字体替换注册表，建议重启

### Bottles D盘映射问题
- U盘未自动映射到 Bottles 的 D 盘（空文件夹）
- 修复方式一：`winecfg` 添加 D 盘映射 → `/media/claw/_ز_`
- 修复方式二：`flatpak override --filesystem=/media` 授予 Flatpak 权限
- 最终用户反馈重启后仍看不到 → 需要进一步排查
- 缺失原因：Flatpak 沙箱限制，虽然已 `flatpak override --filesystem=/media` 但可能还需 `flatpak override --filesystem=$PWD` 或使用真实路径
- 替代方案：在 Bottle 配置中将 D 盘映射到 /media/claw/_ز_ 的完整真实路径（而非 symlink），或通过 winecfg 图形界面手动添加
