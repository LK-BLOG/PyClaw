## 定期清理命令

### 1. 快速清理（安全，推荐）
```bash
# 清理日志、Python缓存
rm -rf logs/* __pycache__
find . -type d -name "__pycache__" -exec rm -rf {} + 2>/dev/null
find . -type f -name "*.pyc" -delete 2>/dev/null
```

### 2. 深度清理（需要时恢复）
```bash
# 清理 Node.js 依赖（Minecraft bot）
rm -rf skills/mc_claw/bot-service/node_modules

# 清理 Python 虚拟环境
rm -rf venv

# 清理 PycLaw 缓存
rm -rf pyclaw/__pycache__ pyclaw-public/__pycache__
```

---

## 恢复方法

### 恢复 Python 虚拟环境
```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

### 恢复 Minecraft bot 依赖
```bash
cd skills/mc_claw/bot-service
npm install
```
