# 🦞 PyClaw 配置说明

## 当前配置

| 项目 | 值 | 说明 |
|------|-----|------|
| API 地址 | `https://api.deepseek.com/v1` | DeepSeek 官方 API (OpenAI 兼容格式) |
| 默认模型 | `deepseek-chat` | DeepSeek 聊天模型 |
| API Key 来源 | `../API.txt` | 从 U 盘根目录读取 |

## 如何修改 API Key

**方法一：修改 U 盘根目录的 API.txt**
```
直接编辑 U 盘根目录下的 API.txt 文件，填入你的 API Key
```

**方法二：修改 webapp.py**
```python
# 编辑 webapp.py，找到 load_api_config() 函数
# 你可以添加更多读取路径或者直接硬编码
```

## 如何切换到其他模型

**编辑 `webapp.py`，找到：**
```python
gateway = Gateway(
    llm_api_key=api_key,
    base_url="https://api.deepseek.com",  # 修改这里
    model="deepseek-chat"                  # 修改这里
)
```
