# 🦞 PyClaw 配置与使用指南

PyClaw 是一个轻量级 Python AI 助手，支持桌面和 Web 双模式。你可以用它来聊天、写代码、做 PPT、剪视频、查天气、搜信息…… 所有功能通过对话驱动。

---

## 🚀 快速安装

一行命令，从零到运行：

| 平台 | 命令 |
|------|------|
| 🐧 Linux/macOS | `curl -fsSL https://raw.githubusercontent.com/LK-BLOG/PyClaw/main/install.sh \| bash` |
| 🪟 PowerShell | `iwr -useb https://raw.githubusercontent.com/LK-BLOG/PyClaw/main/install.ps1 \| iex` |
| 🪟 Git Bash/WSL | `curl -fsSL https://raw.githubusercontent.com/LK-BLOG/PyClaw/main/install.sh \| bash` |

安装后：

```bash
pyclaw setup    # 配置向导（API Key、模型、语言）
pyclaw start    # 启动
```

## ⚙️ 配置

### 首次配置 — `pyclaw setup`

运行 `pyclaw setup` 进入交互式配置向导，依次设置：

1. **API Key** — DeepSeek 或 OpenAI 的 API 密钥
2. **Provider** — 模型供应商（DeepSeek / OpenAI / 自定义）
3. **Model** — 具体模型（如 `deepseek-v4-flash`）
4. **Port** — Web 服务端口（默认 2469）
5. **Thinking** — 思考模式（on/off）
6. **Language** — 界面语言（中文 / English）
7. **Endpoint** — API 端点（自动填充，自定义供应商时手动填写）
8. **Skills** — 技能管理（可禁用不需要的技能）

### 查看/修改配置 — `pyclaw config`

```bash
pyclaw config              # 查看全部配置
pyclaw config API_KEY=***  # 设置某项配置
pyclaw config edit         # 用编辑器打开配置文件
```

配置存储在 `pyclaw.json`（工作空间根目录），不会被提交到 Git。

## 🎮 CLI 命令

| 命令 | 说明 |
|------|------|
| `pyclaw setup` | 配置向导 |
| `pyclaw start` | 启动（交互选择模式） |
| `pyclaw start --mode browser` | 浏览器模式启动 |
| `pyclaw start --mode desktop` | 桌面窗口启动 |
| `pyclaw start --mode background` | 后台服务启动 |
| `pyclaw stop` | 停止 PyClaw |
| `pyclaw status` | 查看运行状态 |
| `pyclaw config` | 查看/设置配置 |
| `pyclaw chat "你好"` | 一句话问答 |
| `pyclaw shell` | 交互式对话（推荐） |

### `pyclaw shell` — 交互式对话

`pyclaw shell` 是推荐的交互方式，支持：

- **多会话** — 启动时列出历史会话，选一个接着聊，或新建会话
- **会话命令**：
  - `/sessions` — 列出所有历史会话
  - `/session <id>` — 切换到指定会话
  - `/new` — 新建会话
  - `/exit` — 退出
- **Markdown 渲染** — 表格、代码块、标题在终端中格式化显示
- **工具调用** — 执行命令、读写文件、搜索网络等
- **双语界面** — 中文 / English（跟随配置中的 LANGUAGE 设置）

## 🌐 Web 界面

启动后访问 `http://localhost:2469`：

- 左侧栏：对话列表 + 新建
- 右下角：切换主题（暗色/亮色）、切换语言
- 设置面板：切换模型、架构模式（标准/完整/基础）
- 内建工具：代码执行、文件操作、Skill 管理等

## 🧩 开发插件

两种方式：

### 声明式（推荐）

```
skills/your-skill/
└── SKILL.md       ← 纯 Markdown，LLM 直接读取
```

无需 Python 代码，无需注册工具。

### 编程式

```
skills/your-skill/
├── __init__.py    ← Python 类 + ToolDefinition
└── SKILL.md       ← 补充文档
```

见 `skills/ppt/` 示例。
